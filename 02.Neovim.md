## 2. Configuración de Neovim 

 ### (~/.config/nvim/init.lua)
Para Neovim, utilizaremos Lazy.nvim, el gestor de plugins más rápido y moderno actualmente.

```
-- --- Configuración Base (No incluida en las fuentes) ---
vim.g.mapleader = " "
vim.opt.number = true
vim.opt.relativenumber = true
vim.opt.shiftwidth = 2

-- --- Instalación automática de Lazy.nvim ---
local lazypath = vim.fn.stdpath("data") .. "/lazy/lazy.nvim"
if not vim.loop.fs_stat(lazypath) then
  vim.fn.system({ "git", "clone", "--filter=blob:none", "https://github.com/folke/lazy.nvim.git", lazypath })
end
vim.opt.rtp:prepend(lazypath)

-- --- Mejores Plugins para 2026 ---
require("lazy").setup({
  "catppuccin/nvim",             -- Tema premium
  "nvim-treesitter/nvim-treesitter", -- Resaltado de sintaxis avanzado
  "nvim-telescope/telescope.nvim",   -- Búsqueda de archivos borrosa
  "neovim/nvim-lspconfig",           -- Soporte para lenguajes (LSP)
  "williamboman/mason.nvim",         -- Instalador de servidores LSP
  "nvim-neo-tree/neo-tree.nvim",     -- Explorador de archivos lateral
  "nvim-lualine/lualine.nvim",       -- Barra de estado estética
})
```

### (.config/nvim/lua/config/options.lua)

Neovim debe estar organizado para ser mantenible. Este archivo define el comportamiento base:

```
local opt = vim.opt

opt.number = true           -- Mostrar números de línea
opt.relativenumber = true   -- Números relativos para navegación
opt.splitbelow = true       -- Splits horizontales abajo
opt.splitright = true       -- Splits verticales a la derecha
opt.tabstop = 2             -- Espacios por tab
opt.shiftwidth = 2
opt.clipboard = "unnamedplus" -- Usar portapapeles del sistema 
opt.scrolloff = 10          -- Mantener 10 líneas de margen al hacer scroll
```
