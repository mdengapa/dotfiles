## 4. Configuración de Tmux (.config/tmux/tmux.conf)

Esta configuración utiliza TPM (Tmux Plugin Manager) para gestionar plugins y mejorar la productividad. 
Define el prefijo Ctrl+a y la integración con Neovim para navegar entre paneles:

```
# --- Configuración General (No incluida en las fuentes) ---
# Cambiar el prefijo a Control + a (más cómodo)
set -g prefix C-a
unbind C-b
bind C-a send-prefix

# Habilitar el mouse y colores reales
set -g mouse on
set -g default-terminal "tmux-256color"

# --- Mejores Plugins ---
set -g @plugin 'christoomey/vim-tmux-navigator'
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
set -g @plugin 'catppuccin/tmux' # Tema estético
set -g @plugin 'tmux-plugins/tmux-resurrect' # Guardar sesiones, Persistencia tras reinicio
set -g @plugin 'tmux-plugins/tmux-continuum' # Autoguardado

# Ejecutar TPM. Inicializar el gestor de plugins (debe ser la última línea)
run '~/.tmux/plugins/tpm/tpm'
```
