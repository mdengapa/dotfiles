## 4. Configuración de Tmux (.config/tmux/tmux.conf)

Define el prefijo Ctrl+a y la integración con Neovim para navegar entre paneles:

```
# Cambiar prefijo a Ctrl+a
set -g prefix C-a
unbind C-b
bind C-a send-prefix

# Navegación fluida con Neovim
set -g @plugin 'christoomey/vim-tmux-navigator'
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-resurrect' # Persistencia tras reinicio

# Ejecutar TPM
run '~/.tmux/plugins/tpm/tpm'
```
